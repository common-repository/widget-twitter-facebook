Ce plugin affiche deux liens logos pour Twitter et Facebook.
 

== Description ==

<strong>Widget Twitter Facebook</strong> vous permettra de faire apparaitre un lien image pour Twitter et un lien image pour Facebook. Vous pouvez configurer les liens ainsi que si vous voulez ou non Twitter ou Facebook. Vous pouvez egalement configurer le titre du Widget.

== Installation ==

Extraire le fichier zip et d&eacute;posez simplement le contenu dans le r&eacute;pertoire wp-content/plugins/ de votre installation de WordPress, puis activez le plugin depuis la page Plugins.
L installation est automatique si vous passez directement pas votre site et l interface Administrateur
